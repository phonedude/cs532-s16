		<h1 class="single-entry-title">Python para rellenar formularios automáticamente</h1>
		<div class="single-entry-meta">
			<span class="posted-on"><a href="http://freakcode.info/python-para-rellenar-formularios-automaticamente/" rel="bookmark"><time class="entry-date published" datetime="2015-07-26T14:00:52+00:00">26 julio, 2015</time><time class="updated" datetime="2015-07-28T23:27:25+00:00">28 julio, 2015</time></a></span><span class="byline"><span class="author vcard"><a class="url fn n" href="http://freakcode.info/author/vvillalobos/">Victor Villalobos</a></span></span>					</div><!-- .entry-meta -->
	</header><!-- .entry-header -->

				<div class="article-featured-image">
				<img width="752" height="440" src="//freakcode.info/wordpress/wp-content/plugins/a3-lazy-load/assets/images/lazy_placeholder.gif" data-lazy-type="image" data-src="http://i1.wp.com/freakcode.info/wordpress/wp-content/uploads/2015/07/pythbook1.gif?resize=752%2C440" class="lazy lazy-hidden attachment-featured-slider size-featured-slider wp-post-image" alt="pythbook" /><noscript><img width="752" height="440" src="http://i1.wp.com/freakcode.info/wordpress/wp-content/uploads/2015/07/pythbook1.gif?resize=752%2C440" class="attachment-featured-slider size-featured-slider wp-post-image" alt="pythbook" /></noscript>			</div>
			
	<div class="entry-content">
		<p><strong>Python</strong> es un lenguaje maravilloso para casi todo, es flexible y limpio, simple de entender y poderosísimo. La idea es enseñarles un poco como hacer <em>web scrapping </em>con el. Lo anterior es una simple forma bonita de decir &#8220;automatización de relleno de formularios web&#8221;, y de verdad que si es mas bonita. Wikipedia es tu amiga y te lo define mejor:</p>
<blockquote><p>Web scraping es una técnica utilizada mediante programas de software para extraer información de sitios web. Usualmente, estos programas simulan la navegación de un humano en la World Wide Web ya sea utilizando el protocolo HTTP manualmente, o incrustando un navegador en una aplicación.</p>
<p>El web scraping está muy relacionado con la indexación de la web, la cual indexa la información de la web utilizando un robot y es una técnica universal adoptada por la mayoría de los motores de búsqueda. Sin embargo, el web scraping se enfoca más en la transformación de datos sin estructura en la web (como el formato HTML) en datos estructurados que pueden ser almacenados y analizados en una base de datos central&#8230;</p>
<p><a href="https://es.wikipedia.org/wiki/Web_scraping" target="_blank">Wikipedia</a> ~ porque no hay nada que te haga mejor las tareas.</p></blockquote>
<p>¿Para qué alguien quisiera hacer esto? Motivos hay tantos como ideas en el mundo, <del>yo lo uso para spammear sitios y ganar plata(?)</del>&#8230; cada quien verá el por qué. ¿Por qué lo estamos enseñando? ¡Porque siempre es bueno aprender algo nuevo!</p>
<h2>Take me to your code!</h2>
<pre class="line-numbers"><code class="language-python">pip install mechanize

# Nuestro código quedaría asi:

import mechanize
br = mechanize.Browser()
br.open(“http://freakcode.info/formulario_estupido.php”)
br.select_form(nr=1)
br.form[“titulo”]=”Pongamos algún titulo lindo”
br.form[“descripcion”]=”Y acá pongamos todo lo que queremos que valla acá”
br.submit()
print br.response().read()</code></pre>
<p>&nbsp;</p>
<p>Vamos por pasos. Por supuesto lo primero que debemos hacer es  importar la librería recién instalada:</p>
<pre class="line-numbers"><code class="language-python">import mechanize

# Creamos el objeto browser.

br = mechanize.Browser()

# Con este invocamos el método open del objeto browser que creamos anteriormente

br.open(“http://freakcode.info/formulario_estupido.php”)

# Seleccionamos el formulario.

br.select_form(nr=1)

# Ahora lo importante, rellenamos cada campo del formulario usando el form que selecionamos anteriormente.

br.form[“titulo”]=”Pongamos algún titulo lindo”
br.form[“descripcion”]=”Y acá pongamos todo lo que queremos que valla acá”

# Y luego ejecutamos el submit:

br.submit()

# Si necesitan leer la respuesta usamos el método read de la siguiente forma.

print br.response().read()</code></pre>
<p>Y básicamente ya está. Esto es solo el principio, ¡Pueden aprender más si nos siguen!</p>
